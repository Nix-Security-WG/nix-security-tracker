The Nixpkgs Security Tracker is a web service for managing information on vulnerabilities in software distributed through Nixpkgs. It operates on the following distinctions.
  • Untriaged suggestions are automatically generated matches between a CVE and Nixpkgs attributes.
  • Dismissed suggestions are CVEs that already got classified by a human as not affecting Nixpkgs.
  • Draft issues are matches between a CVE and Nixpkgs attributes that got selected as security relevant. They are elected to get published as a GitHub issue, but might need further refinement.
  • Published issues are a read-only mirror of GitHub issues that address one CVE each.
